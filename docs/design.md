## CityMate 前端页面设计需求说明

请为 **CityMate 周末探索多智能体 Demo** 设计并实现一个高完成度的前端页面。整体目标是做出一个适合比赛演示的 **React + TypeScript + Tailwind** 单页应用界面，风格要像真实上线的 SaaS 产品，而不是简单原型。

页面主题是：用户输入周末出行需求，系统通过多智能体协作生成上海周末路线，并在页面中展示 3 条候选路线、地图路径、时间线、预算、风险和重排按钮。

# 15. 最终效果目标

最终页面应该让评委一眼看出：

1. 这是一个完整的 AI 周末路线规划产品。
2. 它不是单纯聊天，而是有多智能体协作流程。
3. 它能根据真实 POI、预算、天气、距离生成路线。
4. 它支持用户反馈后的即时重排。
5. 它的视觉完成度足够高，适合 Demo 展示。

整体视觉应该接近一个成熟产品的首页 Dashboard，重点突出：

* 左侧输入需求
* 中间生成路线
* 右侧地图验证
* 底部质检与重排能力

不要做成普通表单页面，要做成一个有产品感、演示感、智能感的路线规划工作台。
